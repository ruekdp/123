# Love Language Gift Recommender

Сервис подбора подарков на основе "языка любви" с интеграцией ALEM PLUS AI. 

## Особенности
- **Telegram Bot**: Пошаговая анкета с поддержкой трех языков (KZ, RU, EN).
- **Web App**: Современный интерфейс на React + Vite с Glassmorphism дизайном.
- **Интеллектуальный подбор**: Использование AlemLLM для анализа эмоциональных потребностей.
- **Маркетплейсы**: Автоматическая генерация ссылок на Kaspi, Wildberries и Ozon.
- **Фото**: Визуализация идей через Unsplash API.

## Запуск проекта
1. Установите зависимости: `npm install`
2. Настройте `.env` файл (BOT_TOKEN и API ключи).
3. Запуск веб-приложения: `npm run dev`
4. Запуск бота: `node bot.js`
